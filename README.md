# ImageToExcel_IDX Releases

이 저장소는 ImageToExcel_IDX의 배포 ZIP 파일과 자동 업데이트 정보를 제공하기 위한 Public 저장소입니다.

소스 코드는 Private 저장소에서 관리합니다.
이 저장소에는 소스 코드, 개발 파일, 인증 정보, 고객사 자료를 올리지 않습니다.

## 구성

- updates.json: 자동 업데이트 확인용 버전 정보 파일
- GitHub Releases: 버전별 배포 ZIP 파일
